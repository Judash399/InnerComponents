# Tutorials

Welecome to the *InnerComponents* Tutorials! These Guides will help you understand how to use the library.

!!! warning "Keep in mind!"
    *InnerComponents* is in a pre-release state alongside *Fusion*. Beacuse of this, features may be added or removed, and logic may become unusable with each update.

    *InnerComponents* will always stay up to date with the newest *Fusion* version. Whenever a newer version of *Fusion* comes out, the next verion of *InnerComponents* will then adopt that version.

    Thank you for using *InnerComponents*!

## Previous Knowledge
It is assumed you know the following subjects.

* Understanding of the Luau language.
* Understanding of the [Fusion UI library](https://elttob.uk/Fusion/).
